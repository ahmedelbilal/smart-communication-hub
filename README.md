## Getting Started

1. **Download and extract** the desired `.zip` file (backend or frontend).
2. Open the extracted folder.
3. Follow the setup guide in its included `README.md` file.

Example:

```bash
unzip smart-communication-hub-api-main.zip
cd smart-communication-hub-api-main
cat README.md
```

## About the Project

**Smart Communication Hub** is a full-stack application that combines:

* Real-time messaging via WebSockets
* AI-powered conversation insights
* Secure backend with NestJS and PostgreSQL
* Modern frontend for seamless interaction
